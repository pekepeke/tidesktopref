Titanium.UI.UserWindow

object
0.2.0
An object representing a top-level Titanium window.

Methods

close Closes a window
createWindow Creates a new window as a child of the current window
focus Focuses a window
getBounds Return this window's bounds object. A bounds object is a simple JavaScript object containing four properties x, y, width, and height which correspond to the window geometry on the screen in pixels.
getChildren Get all children of this UI.UserWindow. All windows open from the context of this window are considered children. When a window is closed all of its children will also be closed automatically.
getContextMenu Return the context menu set on this UI.Userwindow or null if none is set.
getDOMWindow Return the WebKit DOMWindow of the page loaded in this window if one exists, otherwise return null. A DOMWindow object will not be available until a UI.UserWindow's PAGE_INITIALIZED event has fired.
getHeight Return this window's height in pixels.
getID Return this window's configuration id.
getIcon Return this window's icon, if one is set or null
getMaxHeight Return this window's maximum height.
getMaxWidth Return this window's maximum height in pixels.
getMenu Return the window menu set on this UI.UserWindow if one is set, otherwise return null.
getMinHeight Return this window's minimum height.
getMinWidth Return this window's minimum width.
getParent Return this window's parent window or null if it is a top-level window.
getTitle Return the title of this window.
getTransparency Return this window's opacity.
getURL Return this window's current URL."
getWidth Return this window's width in pixels.
getX Return a UI.UserWindow's horizontal (X-axis) position on the screen. The origin of the screen is considered to be the top-left on all platforms.
getY Return a UI.UserWindow's vertical (Y-axis) position on the screen. The origin of the screen is considered to be the top-left on all platforms.
hasTransparentBackground Checks whether a window has a transparent background or not. If a window has a transparent background, transparent colors on the page will show through to windows underneath.
hide Hides a window
isActive Return true if this window is active. An active window is one that has finished opening, but has not yet been closed.
isCloseable Checks whether a window could be closed or not
isDocumentEdited Checks whether a window is in an edited state
isFullscreen Checks whether a window is in fullscreen
isMaximizable Checks whether a window could be maximized or not
isMaximized Checks whether a window is maximized
isMinimizable Checks whether a window could be minimized or not
isMinimized Checks whether a window is minimized
isResizable Checks whether a window is resizable
isTopMost Checks whether a window is top most
isUsingChrome Checks whether a window uses system chrome
isVisible Checks whether a window is visible
maximize Maximizes a window
minimize Minimizes a window
open Opens a window
openFileChooserDialog Displays the file chooser dialog
openFolderChooserDialog Displays the folder chooser dialog
openSaveAsDialog Displays the save as file dialog. Available options: title: string to use for dialog title path: path to where the dialog should be opened at types: list of file extensions that are allowed to be selected multiple: if true, allow user to select more than one file [default: true] defaultFile: default name to be used for saving
setBounds Set this window's bounds object. A bounds object is a simple JavaScript object containing four properties x, y, width, and height which correspond to the window geometry on the screen in pixels.
setCloseable Sets whether a window could be closed or not
setContents Set the contents of the UserWindow, given an HTML string and a base URL. Relative links in the HTML will be resolved relatively to the base URL.
setContextMenu Set this window's context menu
setDocumentEdited Set a window to the edited (a dot in the close button) or unedited state. OS X only.
setFullscreen Makes a window fullscreen
setHeight Sets a window's height
setIcon Sets a window's icon
setMaxHeight Sets a window's max-height
setMaxWidth Sets a window's max-width
setMaximizable Sets whether a window could be maximized or not
setMenu Set this window's menu
setMinHeight Sets a window's min height
setMinWidth Sets a window's min-width
setMinimizable Sets whether a window could be maximized or not
setPluginsEnabled Set if plugins are enabled
setResizable Sets whether a window could be resized or not
setTitle Sets the title of a window
setTopMost Sets whether a window is top most (above other windows)
setTransparency Sets a window's transparency value
setURL Sets the url for a window
setUsingChrome Sets whether a window should use system chrome
setVisible Sets the visibility of the window
setWidth Sets a window's width
setX Set a UI.UserWindow's horizontal (X-axis) position on the screen. The origin of the screen is considered to be the top-left on all platforms.
setY Set a UI.UserWindow's vertical (Y-axis) position on the screen. The origin of the screen is considered to be the top-left on all platforms.
show Shows a window
showInspector Show a UI.UserWindow's web inspector.
unfocus Unfocuses a window
unmaximize Unmaximizes a window
unminimize Unminimizes a window
