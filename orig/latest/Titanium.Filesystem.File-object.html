Titanium.Filesystem.File

object
0.2.0
An object which represents a path to a file or directory. A Filesystem.File is not guaranteed to be a valid path.

Methods

copy Copies a file to the specified location. If the target is a directory the file will be written to that directory. If the target is a file, the file will be written to that file.
createDirectory Create a new directory at this File object's path. If the directory already exists, this method will do nothing. This method will return true if the directory was created or false if it was not.
createShortcut Creates a shortcut to a file or directory at this Filesystem.File's path. On Windows this will create a Win32 style shortcut. On Linux and OS X this will create a symlink. If the operation is successful this method will return true and false otherwise.
createTimestamp Return the timestamp for when this file or directory was created. The return value of this function will be microseconds since the epoch at the time this file was created.
deleteDirectory Remove the file or directory specified by this Filesystem.File. Return true if removal succeeded and false otherwise.
deleteFile Remove the file or directory specified by this Filesystem.File, this method never removes directories recursively. Return true if removal succeeded and false otherwise.
exists Return true if a file or directory exists at the path specified by this Filesystem.File.
extension Return the extension of this Filesystem.File.
getDirectoryListing If this Filesystem.File specifies the path to a directory, return an Array of items inside this directory. If this path does not exist or is not a directory, return null.
isDirectory Return true if the path specified by this Filesystem.File refers to a directory or false otherwise.
isExecutable Return true if the directory or file at the path specified by this Filesystem.File object has the executable bit set. If the file does not exists, this method will return false.
isFile Return true if the path specified by this Filesystem.File refers to a file or false otherwise.
isHidden Return true if the path specified by this Filesystem.File refers to a hidden file. On Windows this means that the file has the hidden attribute and on Linux and OS X this means that the basename of this path begins with a period.
isReadonly Return true if the path specified by this Filesystem.File refers to a read-only file.
isSymbolicLink Return true if the path specified by this Filesystem.File refers to a symbolic link.
isWritable Return true if the path specified by this Filesystem.File refers to a writable file and false otherwise.
modificationTimestamp Return the timestamp for when this file or directory was modified. The return value of this function will be microseconds since the epoch at the last time this file was modified.
move Moves to the specified location. If the target is a directory the file will be written to that directory. If the target is a file, the file will be written to that file.
nativePath Return the full path of this File.Filesystem object.
open Create and open a Filestream for this File object.
parent Return the parent directory of this File.Filesystem object.
read Return the entire contents of a file as a Bytes object.
readLine Return one line from the file specified by this Filesystem.File object. This method will return null when it finishes reading the file or if it encounters and error. The file object will remain in an open state until the entire file has been read via Filestream.File.readLine. It is recommended that new applications use Filesystem.Filestream to read files instead, which allow for explicit control over the native file handles.
rename Renames a file to the given path.
resolve Resolves a given relative path against the path specified by this Filesystem.File object. If the given path is absolute, the absolute path is returned.
setExecutable Set the execute bit on the file or directory at the path specified by this Filesystem.File object. Return true if the file or directory at this path is executable after the operation completes.
setReadonly Remove the wrtiable bit and ensure there is a readable bit on the file or directory at the path specified by this Filesystem.File object. Return true if the file or directory at this path is read-only after the operation completes.
setWritable Set the write bit on the file or directory at the path specified by this Filesystem.File object. Return true if the file or directory at this path is writable after the operation completes.
size Return the size of the file in number of bytes.
spaceAvailable Return the space available on the filesystem containing the path specified by this Filesystem.File object in number of bytes..
toString Return the path of this Filesystem.File object as a String.
touch Creates a new, empty file in an atomic operation. Returns true if the file was created or false if it already exists.
unzip If this file is a zip file, unzip it into the given destination directory. Return true if the operation was successful or false otherwise. This operation will block the main thread.
write Write data to the file at the path specified by this Filesystem.File object. Returns true if any data was written to the file or false otherwise.
